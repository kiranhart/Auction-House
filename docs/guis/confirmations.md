# Confirmation GUIs

Confirmation GUIs provide safety checks before performing important actions.

## Confirmation Types

### Purchase Confirmation

Shown when purchasing items:

-   **Item Preview**: See what you're buying
-   **Price Display**: Total cost
-   **Seller Info**: Who you're buying from
-   **Confirm/Cancel**: Confirm or cancel purchase

### Bid Confirmation

Shown when placing bids:

-   **Item Preview**: Item you're bidding on
-   **Bid Amount**: Your bid amount
-   **Current Bid**: Current highest bid
-   **Total Cost**: Total amount (if money held)
-   **Confirm/Cancel**: Confirm or cancel bid

### Listing Confirmation

Shown when creating listings:

-   **Item Preview**: Item you're listing
-   **Price Info**: All pricing details
-   **Time Info**: Listing duration
-   **Options**: All selected options
-   **Confirm/Cancel**: Confirm or cancel listing

### Cancel Confirmation

Shown when canceling listings:

-   **Item Preview**: Item being canceled
-   **Warning**: Warning about cancellation
-   **Restrictions**: Any restrictions
-   **Confirm/Cancel**: Confirm or cancel

### General Confirmation

For other important actions:

-   **Action Description**: What will happen
-   **Details**: Relevant details
-   **Warning**: Any warnings
-   **Confirm/Cancel**: Confirm or cancel

## Confirmation Features

### Safety Checks

-   **Prevent Mistakes**: Avoid accidental actions
-   **Review Details**: Check before confirming
-   **Cancel Option**: Can always cancel
-   **Time Limit**: May have time limit

### Information Display

-   **Full Details**: Complete information
-   **Cost Display**: All costs shown
-   **Warnings**: Important warnings
-   **Restrictions**: Any restrictions

## Confirmation Process

### Standard Flow

1. **Action Initiated**: Start an action
2. **Confirmation Shown**: Confirmation GUI appears
3. **Review**: Review all information
4. **Confirm or Cancel**: Make decision
5. **Action Executed**: Action proceeds if confirmed

### Time Limits

Some confirmations have time limits:

-   **Countdown**: Time remaining shown
-   **Auto-Cancel**: Cancels if time expires
-   **Extend**: May be able to extend time

## Tips

1. **Read Carefully** - Review all information
2. **Check Prices** - Verify prices before confirming
3. **Review Options** - Check all selected options
4. **Use Cancel** - Cancel if unsure
5. **Don't Rush** - Take time to review

-   `/ah confirm` - Confirm pending actions
