# Main Auction House GUI

The main auction house GUI is the primary interface for browsing all available auctions. It provides comprehensive filtering, sorting, and search capabilities.

## Accessing the GUI

Open the main auction house GUI using:
```
/ah
```

Or with search keywords:
```
/ah <keywords>
```

## GUI Layout

### Item Display Area
- **Primary Area**: Displays all available auction listings
- **Pagination**: Multiple pages if many listings exist
- **Item Slots**: Configurable slot positions for items

### Navigation Buttons
- **Previous Page**: Navigate to previous page
- **Next Page**: Navigate to next page
- **Page Indicator**: Shows current page and total pages

### Filter Buttons
- **Filter Menu**: Access to filtering options
- **Category Filters**: Filter by item category
- **Price Filters**: Filter by price range
- **Seller Filters**: Filter by seller name
- **Material Filters**: Filter by material type

### Action Buttons
- **Your Listings**: Quick access to your active/expired listings
- **Shopping Cart**: Access to your cart
- **Search**: Search functionality
- **Settings**: Personal settings (if available)

## Item Display

Each auction item shows:
- **Item Preview**: Visual representation of the item
- **Item Name**: Display name of the item
- **Price**: Current price or highest bid
- **Seller**: Name of the seller
- **Time Remaining**: Countdown until expiration
- **Bid Count**: Number of bids (for auctions)
- **Listing Type**: Auction or BIN indicator

## Interactions

### Viewing Items
- **Left Click**: View item details
- **Right Click**: Quick purchase/bid (if enabled)
- **Shift Click**: Alternative actions

### Purchasing Items
- **Buy Button**: Purchase BIN listings instantly
- **Bid Button**: Place a bid on auction listings
- **Add to Cart**: Add item to shopping cart

### Item Details
Clicking an item opens a detailed view showing:
- Full item information
- Enchantments and attributes
- Seller details
- Bid history (for auctions)
- Purchase/bid options

## Filtering

### Category Filters
Filter items by category:
- Weapons
- Armor
- Tools
- Blocks
- Food
- And more (configurable)

### Price Filters
- **Minimum Price**: Show items above a price
- **Maximum Price**: Show items below a price
- **Price Range**: Set both min and max

### Other Filters
- **Seller Filter**: Show items from specific sellers
- **Material Filter**: Filter by material type
- **Enchantment Filter**: Filter by enchantments
- **Custom Filters**: User-created filter sets

## Sorting

Sort listings by:
- **Price**: Low to high or high to low
- **Time**: Time remaining (soonest first or latest first)
- **Seller**: Alphabetically by seller name
- **Bids**: Number of bids (auctions only)
- **Newest**: Most recently listed first
- **Oldest**: Oldest listings first

## Search

The GUI includes search functionality:
- **Search Bar**: Type keywords to search
- **Real-time Filtering**: Results update as you type
- **Search History**: Recent searches (if enabled)

## Auto-Refresh

If enabled, the GUI automatically refreshes:
- **Update Interval**: Configurable refresh rate
- **Real-time Updates**: New listings appear automatically
- **Price Updates**: Bid prices update in real-time

## Navigation Tips

1. **Use Filters** - Narrow down results quickly
2. **Sort Options** - Find items by what matters to you
3. **Search Feature** - Use for specific items
4. **Pagination** - Navigate through multiple pages
5. **Quick Actions** - Use right-click for faster actions
