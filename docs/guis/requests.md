# Requests GUI

The requests GUI allows you to create and manage item requests.

## Accessing the GUI

Open requests using:
```
/ah request
```

Or create a request directly:
```
/ah request <price>
```

## Creating a Request

### Request Creation GUI

When you use `/ah request`, a GUI opens for:
- **Item Selection**: Select the item you're holding
- **Price Input**: Set maximum price you'll pay
- **Quantity**: Set desired quantity
- **Duration**: Set request expiration time
- **Create**: Confirm request creation

### Request Requirements
- **Item in Hand**: Must be holding an item
- **Valid Price**: Price must meet requirements
- **Not at Limit**: Must not exceed request limits
- **Not Banned**: Must not be banned from requests

## Request Display

### Your Requests
- **Active Requests**: Shows your active requests
- **Request Status**: Whether fulfilled or pending
- **Price Display**: Maximum price you set
- **Time Remaining**: Time until expiration

### Fulfilled Requests
- **Fulfillment Notice**: When someone fulfills
- **Purchase Option**: Buy the fulfilled item
- **Seller Info**: Who fulfilled the request
- **Price Match**: Price matches your request

## Fulfilling Requests

Other players can fulfill your requests:
- **See Requests**: Requests appear in auction house
- **List Items**: Players list items to fulfill
- **Notification**: You're notified when fulfilled
- **Purchase**: Buy the item immediately

## Managing Requests

### View Requests
- **Active Requests**: See all your requests
- **Fulfilled Requests**: See fulfilled requests
- **Expired Requests**: See expired requests

### Cancel Requests
- **Cancel Button**: Cancel a request
- **Restrictions**: May have restrictions
- **Confirmation**: May require confirmation

## Request Information

### Displayed Information
- **Item**: Item being requested
- **Price**: Maximum price
- **Quantity**: Desired quantity
- **Status**: Fulfilled or pending
- **Time**: Time remaining

## Tips

1. **Set Reasonable Prices** - Don't set too low
2. **Be Specific** - Request specific items
3. **Monitor Requests** - Check if fulfilled
4. **Cancel Unwanted** - Remove requests you don't need
5. **Use for Rare Items** - Great for hard-to-find items

## Related Commands

- `/ah request` - Create requests
- `/ah active` - View your requests
- `/ah expired` - View expired requests
