# Filter GUIs

The filter GUIs allow you to configure and manage item filters for the auction house.

## Filter Selection GUI

Accessed when clicking filter buttons in the main auction house.

### Filter Options
- **Category Filters**: Filter by item category
- **Price Filters**: Filter by price range
- **Seller Filters**: Filter by seller
- **Material Filters**: Filter by material
- **Enchantment Filters**: Filter by enchantments
- **Custom Filters**: User-created filters

### Filter Types
- **Whitelist**: Show only these items
- **Blacklist**: Hide these items
- **Price Range**: Min/max price
- **Combination**: Multiple filters together

## Filter Whitelist GUI

Accessed via `/ah filter`.

### Features
- **View Whitelist**: See whitelisted items
- **Add Items**: Add items to whitelist
- **Remove Items**: Remove from whitelist
- **Category Management**: Manage by category

### Adding Items
1. **Hold Item**: Hold the item to add
2. **Use Command**: `/ah filter additem <category>`
3. **Or GUI**: Add through GUI interface
4. **Confirm**: Item added to whitelist

## Filter Configuration

### Category Filters
- **Weapons**: Filter weapons
- **Armor**: Filter armor
- **Tools**: Filter tools
- **Blocks**: Filter blocks
- **Food**: Filter food items
- **And More**: Additional categories

### Price Filters
- **Minimum Price**: Show items above price
- **Maximum Price**: Show items below price
- **Price Range**: Set both min and max

### Other Filters
- **Seller Filter**: Filter by seller name
- **Material Filter**: Filter by material type
- **Enchantment Filter**: Filter by enchantments
- **Custom Filters**: Create custom filter sets

## Filter Management

### Saving Filters
- **Save Filter Set**: Save current filters
- **Load Filter Set**: Load saved filters
- **Delete Filter Set**: Remove saved filters
- **Default Filters**: Use default filters

### Filter Persistence
- **Remember Filters**: Filters may persist
- **Reset Filters**: Clear all filters
- **Default State**: Return to defaults

## Using Filters

### Applying Filters
1. **Select Filters**: Choose filter options
2. **Apply**: Apply filters to listings
3. **View Results**: See filtered results
4. **Adjust**: Modify filters as needed

### Filter Effects
- **Narrow Results**: Reduce number of items
- **Focus Search**: Find specific items
- **Save Time**: Faster item finding
- **Custom Views**: Personalized views

## Tips

1. **Use Multiple Filters** - Combine filters for better results
2. **Save Filter Sets** - Save commonly used filters
3. **Reset When Needed** - Clear filters to see all items
4. **Experiment** - Try different filter combinations
5. **Use Whitelist** - Whitelist favorite item types

## Related Commands

- `/ah filter` - Manage filters
- `/ah` - Apply filters in main GUI
