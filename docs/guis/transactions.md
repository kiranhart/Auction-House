# Transactions GUI

The transactions GUI provides a comprehensive view of all your auction house activity.

## Accessing the GUI

Open transactions using:
```
/ah transactions
```

Or search for specific player:
```
/ah transactions search <player>
```

## GUI Layout

### Transaction Type Selection
When first opened, you can choose:
- **All Transactions**: View everything
- **Purchases**: Items you've bought
- **Sales**: Items you've sold
- **Bids**: Bids you've placed
- **Payments**: Payments received

### Transaction List
- **Chronological Display**: Shows transactions in order
- **Transaction Details**: Full information for each
- **Filtering Options**: Filter by type, date, player
- **Search**: Search transactions

## Transaction Types

### Purchases
- Items you've purchased
- Purchase price
- Seller information
- Purchase date

### Sales
- Items you've sold
- Sale price
- Buyer information
- Sale date

### Bids
- Bids you've placed
- Bid amounts
- Auction information
- Bid status (won/lost/active)

### Payments
- Payments received
- Payment amounts
- Payment reasons
- Payment dates

## Transaction Information

### Displayed Details
- **Type**: Transaction type indicator
- **Amount**: Transaction value
- **Item**: Item involved
- **Other Party**: Buyer/seller/bidder
- **Date**: When it occurred
- **Status**: Transaction status

## Filtering

### Filter Options
- **Type Filter**: Filter by transaction type
- **Date Range**: Filter by date
- **Player Filter**: Filter by player
- **Amount Range**: Filter by amount
- **Item Filter**: Filter by item type

### Search
- **Keyword Search**: Search by keywords
- **Player Search**: Search by player name
- **Item Search**: Search by item name
- **Date Search**: Search by date

## Viewing Transactions

### Transaction Details
Click a transaction to see:
- **Full Details**: Complete information
- **Item Preview**: Item involved
- **Other Party**: Buyer/seller details
- **Timeline**: Transaction timeline
- **Related**: Related transactions

## Exporting

Some servers support exporting:
- **Export Data**: Export transaction history
- **CSV Format**: Export as CSV
- **Date Range**: Export specific dates
- **Filtered Export**: Export filtered results

## Tips

1. **Review Regularly** - Check your transaction history
2. **Use Filters** - Narrow down results
3. **Search Feature** - Find specific transactions
4. **Track Activity** - Monitor your auction activity
5. **Export Data** - Keep records if needed
