# GUIs

Auction House provides a comprehensive GUI-based interface for all auction operations. All interactions are done through intuitive graphical interfaces.

## GUI Overview

Auction House uses GUIs for:
- Browsing auctions
- Listing items for sale
- Placing bids
- Managing your auctions
- Admin functions
- Configuration

## Main GUIs

### [Main Auction House GUI](guis/main-auction-house.md)
The primary interface for browsing all available auctions. Features filtering, sorting, pagination, and search.

### [Selling GUIs](guis/selling.md)
Complete set of GUIs for listing items:
- Listing type selection
- Item placement
- Auction configuration
- BIN configuration
- Bundle creation

### [Bidding GUIs](guis/bidding.md)
GUIs for managing bids:
- Active bids view
- Bid placement interface
- Bid management

### [Cart GUI](guis/cart.md)
Shopping cart interface for managing multiple items before purchase.

### [Expired Items GUI](guis/expired-items.md)
View and collect items from expired listings.

### [Active Auctions GUI](guis/active-auctions.md)
View and manage your currently active listings.

### [Payments GUI](guis/payments.md)
Collect payments from sold items.

### [Transactions GUI](guis/transactions.md)
View complete transaction history with filtering options.

### [Requests GUI](guis/requests.md)
Create and manage item requests.

### [Statistics GUIs](guis/statistics.md)
View statistics and leaderboards.

### [Admin GUIs](guis/admin.md)
Administrative interfaces for managing the auction house.

### [Filter GUIs](guis/filters.md)
Configure and manage item filters.

### [Confirmation GUIs](guis/confirmations.md)
Confirmation dialogs for important actions.

## GUI Features

### Navigation
- **Previous/Next Buttons** - Navigate between pages
- **Page Indicators** - See current page and total pages
- **Quick Navigation** - Jump to specific pages

### Filtering
- **Category Filters** - Filter by item category
- **Price Filters** - Filter by price range
- **Seller Filters** - Filter by seller
- **Material Filters** - Filter by material type
- **Custom Filters** - Create custom filter sets

### Sorting
- **Price** - Sort by price (low to high, high to low)
- **Time** - Sort by time remaining
- **Seller** - Sort alphabetically by seller
- **Bids** - Sort by number of bids (auctions only)

### Search
- **Keyword Search** - Search by item name
- **Description Search** - Search item descriptions
- **Real-time Filtering** - Results update as you type

### Item Display
- **Item Preview** - See item details
- **Price Display** - Current price or highest bid
- **Time Remaining** - Countdown timer
- **Seller Info** - Seller name and details
- **Bid Information** - Bid count and history

## GUI Interactions

### Click Actions
- **Left Click** - Select/view item
- **Right Click** - Quick actions
- **Shift Click** - Alternative actions
- **Middle Click** - Special functions

### Item Actions
- **Buy** - Purchase item (BIN listings)
- **Bid** - Place a bid (auction listings)
- **Add to Cart** - Add to shopping cart
- **View Details** - See full item information
- **Inspect** - Inspect item (containers, shulker boxes)

### Navigation Actions
- **Previous Page** - Go to previous page
- **Next Page** - Go to next page
- **First Page** - Jump to first page
- **Last Page** - Jump to last page
- **Close** - Close GUI

## GUI Customization

Many GUI elements can be customized:
- **Titles** - Custom GUI titles
- **Layouts** - Item slot positions
- **Buttons** - Custom button designs
- **Colors** - Color schemes
- **Items** - Decorative items

## Auto-Refresh

Some GUIs support auto-refresh:
- **Main Auction House** - Updates automatically
- **Active Auctions** - Real-time updates
- **Cart** - Updates when items change

## GUI Tips

1. **Use Filters** - Filter results to find items faster
2. **Sort Options** - Sort by what matters to you
3. **Search Feature** - Use search for specific items
4. **Cart Feature** - Add multiple items before purchasing
5. **Keyboard Shortcuts** - Some GUIs support keyboard navigation
